A. 차 안 콘텐츠 (자율주행 중)

FSD가 운전하는 동안 차 안에서 펼쳐지는 크리에이티브 콘텐츠 10가지. 핵심은 "핸즈프리"의 자유로움을 시각적으로 증명하는 것.

A1

자율주행 중 풀메이크업 챌린지

自动驾驶全妆挑战
🎬 컨셉 상세

FSD를 켜고 서울→강릉 영동고속도로에서 와이프가 풀 메이크업을 처음부터 끝까지 완성한다. 남편은 핸들을 잡지 않고 옆에서 리액션하며, 차가 스스로 차선 변경하거나 앞차를 추월하는 순간에 카메라를 FSD 화면과 와이프 얼굴 사이를 왔다갔다 한다. 메이크업 전 민낯과 완성 후의 극적인 비교가 핵심이다. 강릉 도착 시 완벽한 메이크업으로 카페에 등장하는 것이 마무리. 베이스부터 립까지 전 과정을 보여주되, FSD 개입 순간(차선 변경, 커브)마다 "지금 자율주행 중" 자막을 삽입해 긴장감을 유지한다.

📸 촬영 포인트
  • 대시보드 고정 카메라: FSD 화면(비전 뷰) 실시간 녹화 — 차선·차량 인식 그래픽이 보이도록
  • 조수석 클로즈업 카메라 (GoPro/인스타360): 와이프 얼굴 정면, 메이크업 디테일
  • 운전석 사이드 카메라: 남편 핸즈프리 상태 + 리액션 포착
  • 타임랩스(x8)로 전체 메이크업 과정 압축 + 차선 변경 순간은 실시간 슬로모션
  • 비포/애프터 스플릿 스크린 — 민낯 좌측, 완성 우측으로 드라마틱 공개
🔥 바이럴 포인트
  • "운전 중 화장"이라는 금기를 자율주행으로 합법적(?)으로 깨는 신선함
  • 민낯→완성 비포애프터 = 중국 뷰티 콘텐츠 황금 포맷 (抖音 조회수 보장)
  • FSD 신뢰도를 메이크업이라는 일상 행위로 시각적으로 증명
  • 차선 변경 순간 손 떨림 → 웃음 = 밈 생성 가능성 높음
📍 추천 촬영 장소

영동고속도로 서울→강릉 구간, 특히 대관령 구간 (터널 진입·탈출 시 조명 변화가 영상미 극대화). 출발: 서울 톨게이트 / 도착: 강릉 안목해변 카페거리

🎒 소품 / 준비물

메이크업 풀세트 (파운데이션~립), 차량용 LED 조명 (클립형), 차량용 확대 거울, GoPro Hero 12 + 흡착 마운트, 물티슈, 메이크업 파우치

⏱ 예상 촬영 시간

약 2.5시간 (서울→강릉 이동 중 자연 촬영, 별도 셋업 불필요)

🎵 추천 BGM

메이크업 타임랩스: 경쾌한 팝 (Dua Lipa - Levitating 분위기) → 완성 reveal: 드라마틱 오케스트라 스웰 → 카페 등장: 세련된 Lo-fi

A2

고속도로 패션쇼 — 옷 갈아입기 타임랩스

高速公路时装秀 — 换装延时摄影
🎬 컨셉 상세

FSD 자율주행 중 와이프가 조수석에서 도시별 컨셉에 맞는 의상으로 갈아입는다. 서울 출발 시 캐주얼 스트릿 룩 → 강릉 도착 전 내추럴 리조트 룩 → 부산 전 글래머러스 나이트 룩으로 변신. 각 의상 교체는 터널 진입 시 시작해서 터널 탈출 시 완성되는 "터널 트랜지션"으로 편집한다. 남편은 FSD 덕분에 완전히 뒤를 돌아 와이프의 코디를 평가하는 리액션을 보여준다. 마지막에는 세 벌을 나란히 비교하는 스플릿 화면으로 마무리. 차 안이라는 좁은 공간에서 옷을 갈아입는 과정 자체가 코미디 요소가 된다.

📸 촬영 포인트
  • 터널 진입 → 화면 어두워짐 → 터널 탈출 → 새 의상 공개 (트랜지션 편집)
  • 조수석 정면 고정캠 + 백미러 앵글에서 동시 촬영
  • 옷 교체 과정은 유쾌한 빠른 컷으로, 좁은 공간에서의 우스꽝스러움 강조
  • 각 룩 완성 후 차 창문을 배경으로 포즈 — 달리는 풍경이 자연 배경
  • 3단 비교 컷: 같은 앵글에서 세 벌의 룩을 연속 배치
🔥 바이럴 포인트
  • "터널 트랜지션"은 틱톡/도우인에서 이미 검증된 인기 편집 기법
  • 패션 콘텐츠 + 여행 콘텐츠 + 기술 콘텐츠의 3중 카테고리 노출
  • 좁은 차 안에서 옷 갈아입는 코미디 = 공감 + 웃음
  • 각 도시에 맞는 패션으로 여행 가이드 역할까지
📍 추천 촬영 장소

영동고속도로 → 동해고속도로 구간. 터널이 많은 대관령~강릉 구간이 최적. 강릉→부산 동해안 루트의 해안 터널도 활용.

🎒 소품 / 준비물

3벌의 컨셉 의상 (스트릿/리조트/글램), 액세서리 세트별 분류, 의상 커버백, 차량용 옷걸이, 소형 스팀다리미(주름 방지)

⏱ 예상 촬영 시간

총 여정 중 연속 촬영 약 4시간 (3회 의상 교체, 각 20분 소요)

🎵 추천 BGM

런웨이 느낌 일렉트로닉 비트 (터널 트랜지션 시 베이스 드롭) → 각 룩 공개 시 해당 도시 무드 음악

A3

자율주행 먹방 — 편의점 풀코스

自动驾驶吃播 — 便利店全套大餐
🎬 컨셉 상세

고속도로 휴게소에서 편의점 음식을 풀코스로 구매한 뒤, FSD를 켜고 달리면서 차 안에서 정식 코스 요리처럼 세팅하여 먹는다. 전채(삼각김밥) → 수프(컵라면) → 메인(도시락) → 디저트(편의점 케이크) 순서로 남편이 웨이터 역할을 하며 서빙한다. 차가 자율주행 중이라 두 사람 모두 손이 자유롭다는 것을 강조. 테이블 냅킨, 와인잔(에 콜라)까지 세팅해 B급 유머를 극대화한다. 와이프는 마치 미쉐린 레스토랑인 듯 품평하고, 남편은 정장 앞치마를 두르고 진지하게 요리를 설명한다.

📸 촬영 포인트
  • 센터콘솔 위 미니 테이블 세팅 — 오버헤드 앵글로 음식 배치 촬영
  • 남편 서빙 모습: 좁은 차 안에서 우아한 제스처 → 코미디
  • 와이프 맛 리액션 클로즈업 + FSD 화면 PIP(Picture-in-Picture)
  • 창밖 풍경 속도감 + 차 안의 고급(?) 분위기 대비
  • 각 코스 전환 시 "다음 코스" 자막 + 벨 소리 효과음
🔥 바이럴 포인트
  • 한국 편의점 음식 = 중국에서 이미 핫한 콘텐츠 소재
  • B급 유머(편의점 풀코스)는 도우인/샤오홍슈 공유율 최상위
  • "자율주행이니까 가능" = 테슬라 FSD 기능을 자연스럽게 어필
  • 커플 케미(웨이터 남편 + 까다로운 손님 와이프) = 팬 형성
📍 추천 촬영 장소

영동고속도로 여주 휴게소에서 음식 구매 → 강릉 방면 자율주행 구간에서 촬영. 또는 부산→광주 남해고속도로 진주 휴게소 출발.

🎒 소품 / 준비물

미니 접이식 트레이, 테이블 냅킨, 와인잔 2개, 미니 촛대(LED), 정장 앞치마, 편의점 음식 풀코스, 물티슈 다량

⏱ 예상 촬영 시간

약 1.5시간 (구매 30분 + 촬영 1시간)

🎵 추천 BGM

클래식 레스토랑 BGM (비발디 사계 등) → 편의점 음식 등장 시 갑자기 트로트로 전환 (갭 코미디)

A4

FSD 중 커플 네일아트

自动驾驶中情侣美甲
🎬 컨셉 상세

와이프가 자율주행 중인 차 안에서 남편의 손톱에 네일아트를 해준다. 남편은 처음에는 거부하다가 점점 빠져드는 모습을 연출한다. 핵심은 차가 흔들리거나 차선 변경할 때 네일이 삐뚤어지는 순간들 — 이때 와이프의 짜증과 남편의 "차가 그런 거지 내가 그런 거 아니야" 핑계가 웃음 포인트. 완성된 네일에 테슬라 로고를 그려넣는 것이 하이라이트. 마지막에 고속도로 휴게소에서 네일 완성된 손으로 커피를 잡는 컷으로 마무리. 남편이 점점 네일에 진심이 되어가는 과정이 캐릭터 성장(?)

📸 촬영 포인트
  • 손 클로즈업 마크로 렌즈: 네일 작업 디테일 + 차량 진동으로 인한 흔들림
  • 2인 리액션 와이드 샷: 남편 표정 변화 (거부→흥미→진심)
  • 차선 변경 순간 카메라 흔들림 → 네일 삐뚤어짐 → 리액션 3단 콤보
  • 테슬라 로고 그리기 과정은 탑뷰 타임랩스
  • 완성 후 FSD 화면 옆에 네일 손 배치 — 같은 프레임에 기술과 아트
🔥 바이럴 포인트
  • 남자의 네일아트 = 성별 고정관념 깨기 → 진보적 MZ세대 공감
  • 움직이는 차에서 섬세한 작업 = 서스펜스 콘텐츠
  • 테슬라 로고 네일 = 브랜드 팬덤 자극, 테슬라 공식 리포스트 가능성
  • 커플 케미의 극치 — "남편에게 네일해주기" 챌린지로 확산 가능
📍 추천 촬영 장소

강릉→부산 동해고속도로 구간 (해안 도로 풍경이 창밖으로 보이면 영상미 UP). 약 4시간 구간 중 일부 활용.

🎒 소품 / 준비물

젤 네일 키트 (UV 없이 가능한 스티커형 추천), 네일 파일, 스톤/글리터, 가는 붓 (테슬라 로고용), 리무버, 미니 LED 데스크 램프

⏱ 예상 촬영 시간

약 1.5시간 (네일 작업 1시간 + 리액션 촬영)

🎵 추천 BGM

아기자기한 우쿨렐레 팝 → 차선 변경 시 서스펜스 음악 (현악 스팅) → 완성 시 승리의 팡파레

🎬 컨셉 상세

테슬라 트렁크 프러포즈 재현

🎬 컨셉 상세

중국에서 유행하는 "후비샹 구혼(后备箱求婚, 트렁크 프러포즈)"을 테슬라 버전으로 재현한다. 남편이 몰래 테슬라 트렁크를 꽃, 풍선, LED 조명, 사진들로 꾸며놓고 와이프에게 서프라이즈로 보여주는 감동 콘텐츠. 테슬라의 넓은 트렁크 공간과 라이트쇼 기능을 활용해 다른 차에서는 불가능한 연출을 만든다. 이미 결혼한 커플이지만 "프러포즈 재현"이라는 컨셉으로 추억을 되살리며, 와이프의 진짜 감동 리액션을 포착한다. 한국의 야경 명소를 배경으로 밤에 촬영하여 로맨틱한 분위기를 극대화한다.

📸 촬영 포인트
  • 준비 과정 몽타주: 남편이 몰래 트렁크 꾸미는 타임랩스
  • 와이프 등장 장면: 눈 가리고 이동 → 트렁크 오픈 순간 리액션 풀샷
  • 트렁크 내부 클로즈업: 꽃·조명·사진 디테일 ASMR
  • 테슬라 라이트쇼: 외부 조명 퍼포먼스 + 커플 실루엣
  • 드론: 야경 배경 위에서 바라본 조명 빛나는 테슬라 + 커플
🔥 바이럴 포인트
  • 后备箱求婚 = 중국 더우인 억대 조회수 카테고리 (검증된 포맷)
  • 테슬라 라이트쇼 = 다른 차에서 불가능한 차별화 요소
  • 감동 + 로맨스 = 공유·저장 동기 극대화 (여성 시청자 타겟)
  • "결혼 후에도 로맨틱" 메시지 → 기혼 커플 공감 유도

🎒 소품 / 준비물

중국 와이프 vs 한국 남편 간식 대결

中国老婆 vs 韩国老公 零食大战
🎬 컨셉 상세

고속도로 휴게소에 들를 때마다 와이프는 중국식 간식(또는 중국에서 가져온 간식)을, 남편은 한국 간식을 각각 고르고 서로에게 먹여보며 리액션 배틀을 벌인다. 매 라운드마다 점수를 매겨 최종 "간식 챔피언 국가"를 가린다. 테슬라 차 안에서 먹방을 진행하며, FSD 자율주행 중이라 두 사람 모두 편하게 먹을 수 있다는 점을 자연스럽게 보여준다. 매운 과자 라운드, 달콤한 디저트 라운드, 음료 라운드 등으로 구성하고, 마지막에 서로 추천 간식 Top 3를 발표한다. 한중 음식 문화 교류를 가볍고 재미있게 다룬다.

📸 촬영 포인트
  • 차 내부 정면캠: 두 사람 나란히 간식 먹는 모습 (먹방 앵글)
  • 간식 클로즈업: 패키지 + 내용물 ASMR (바삭/쫀득 소리)
  • 리액션 슬로모션: 매운 맛/신맛에 표정 변하는 순간 슬로모
  • 점수판 그래픽: 매 라운드 결과 스코어보드 오버레이
  • 휴게소 입장 장면: 테슬라에서 내려 각자 간식 고르러 뛰어가는 와이드 숏
🔥 바이럴 포인트
  • 한중 간식 비교 = 양국 시청자 모두 관심 + 논쟁(내 나라 간식이 더 맛있어!)
  • 먹방 + 리액션 = 더우인/유튜브 최고 인기 조합
  • 커플 먹방 = 달달함 + 웃음 = 구독 유도
  • 한국 휴게소 문화 소개 → 여행 정보 콘텐츠 역할
📍 추천 촬영 장소

고속도로 유명 휴게소 (안성 휴게소, 횡성 휴게소 등). 다양한 간식이 있는 대형 휴게소 추천

🎒 소품 / 준비물

중국 간식 세트 (미리 구매: 辣条, 旺旺, 大白兔 등), 한국 간식 (허니버터칩, 초코파이, 바나나킥 등), 점수판, 물, 물티슈

⏱ 예상 촬영 시간

약 2시간 (휴게소 쇼핑 30분 + 차 내 먹방 1시간 + 결과 발표 30분)

1도시 1코디 1사진 10분 챌린지

1城市1穿搭1照片 10分钟挑战
🎬 컨셉 상세

각 도시에 도착하면 10분 타이머를 걸고 ① 차에서 코디 체인지(1분) ② 촬영 장소 이동(3분) ③ 인생샷 촬영(5분) ④ 차로 복귀(1분)를 완수하는 스피드 챌린지. 도시마다 그 지역 랜드마크와 어울리는 코디를 미리 준비해두고, 테슬라에서 빠르게 갈아입은 뒤 달려가서 사진을 찍고 돌아온다. 부산 해운대에서는 서퍼룩, 경주에서는 한복, 강릉에서는 카페룩, 전주에서는 레트로룩 등. 10분 안에 완성하지 못하면 벌칙이 있다. 테슬라의 넓은 실내를 '이동식 드레스룸'으로 활용하는 것이 포인트. 최종적으로 모든 도시의 사진을 모아 인스타 캐러셀로 공개한다.

📸 촬영 포인트
  • 화면 상단 10분 카운트다운 타이머 고정
  • 차 안 코디 체인지: 스피드 타임랩스 (코미디 연출)
  • 달리기 구간: 핸드헬드 흔들림캠 (긴박감)
  • 사진 촬영 순간: 갑자기 잡지 화보 톤으로 전환 (느린 셔터 효과음)
  • 전 도시 사진 모아보기: 엔딩에 갤러리 슬라이드쇼
🔥 바이럴 포인트
  • 10분 타이머 = 긴장감 + 숏폼에 딱 맞는 러닝타임
  • 도시별 코디 변신 = 시각적 다양성 → 끝까지 시청 유도
  • "나도 해보고 싶다" 참여 욕구 → 챌린지 확산 가능
  • 최종 사진 갤러리 = 인스타 공유용 콘텐츠 (2차 확산)
📍 추천 촬영 장소

부산 해운대 해변, 경주 첨성대/대릉원, 강릉 안목해변 카페거리, 전주 한옥마을, 서울 경복궁 (5개 도시 시리즈)

🎒 소품 / 준비물

⏱ 예상 촬영 시간

도시당 약 30분 (실제 10분 챌린지 + 서브 앵글·리액션 추가 촬영 20분)
