## 🔴 P0: 핵심 기능 (지금 당장)
> 사용자가 앱을 쓰는 이유. 이게 없으면 앱이 아님.

### 1. 실데이터 연결 — API 연동
범범뻠이 data.go.kr 키 발급하면 즉시 착수. 그 전에 구월이 준비할 것:
- [ ] **날씨/미세먼지 위젯** — 기상청 API + 에어코리아 연동 (Workers 프록시)
- [ ] **지하철 실시간 도착** — 서울 지하철 API → 교통 포켓에 연결
- [ ] **버스 도착정보** — 서울 버스 API → 교통 포켓
- [ ] **병원 검색** — 건강보험심사평가원 API → 의료탭 실데이터
- [ ] **축제/행사** — 한국관광공사 API → 축제 포켓
- [ ] **부동산 시세** — 국토교통부 실거래가/전월세 → 부동산 포켓
- [ ] **법령정보** — law.go.kr API → 비자 관련 법령 (승인 대기 중)
- [ ] **환율 실시간** — 한국은행 API → 환율 위젯 정확도 향상

### 2. 번역 기능 — 핵심 도구
- [ ] **실시간 통역** — DeepSeek API 연동 (범범뻠 키 발급 후)
- [ ] **간판 사전 DB** — 자주 쓰는 한국어 간판 500개+ 중국어 번역
- [ ] **음성 입력** — Web Speech API로 한국어 음성→텍스트→중국어 번역

### 3. 비자 핵심 로직
- [ ] **비자 변경 시뮬레이터** — 현재 비자→목표 비자, 필요 서류/조건 자동 계산
- [ ] **비자 만료 알림** — 실제 푸시 알림 연동
- [ ] **출입국관리사무소 위치/대기시간** — 지도 연동

### 4. 맛집 실데이터
- [ ] **미슐랭 데이터 최신화** — Workers cron으로 자동 갱신 (이미 구축)
- [ ] **카카오맵 연동 강화** — 식당 검색→지도→길찾기 원클릭
- [ ] **블루리본 데이터** — 30개→100개+ 확장

### 5. SOS/긴급
- [ ] **원클릭 긴급전화** — 112/119/1345 바로 연결
- [ ] **현재 위치 공유** — GPS→중국어 주소 변환→긴급 연락처에 전송
- [ ] **증상 표현 카드** — 병원에서 보여줄 수 있는 중한 증상 카드
