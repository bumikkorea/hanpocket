### 안건 007: 온보딩 간소화 (Toss 스타일)

현재 온보딩: splash -> 국적 선택 -> 비자 선택 -> 환율 로딩 -> 완료 (5단계)
변경 후: splash -> 국적+비자 통합 1화면 -> 완료 (2단계)

1) Splash 화면 (1단계):
   - HANPOCKET 로고 + 복주머니
   - "한국 여행의 모든 것" / "韩国旅行的一切" / "Everything for Korea Travel"
   - 1.5초 후 자동 전환 또는 터치 시 전환
   - 배경: 흰색, 미니멀

2) 설정 화면 (2단계):
   - "시작하기" 큰 카드 1개
   - 국적 선택: 드롭다운 (중국대륙/홍콩/마카오/대만/기타)
   - 비자 선택: 드롭다운 (관광/유학/취업/거주/기타)
   - 언어 선택: 3버튼 (한국어/中文/EN)
   - "시작" 버튼 (bg-[#111827] text-white 큰 버튼)
   - "건너뛰기" 링크 (설정 안 해도 진입 가능, 기본값: 중국대륙/관광/zh)

3) 기존 환율 로딩 제거:
   - 온보딩에서 환율 API 호출하지 말 것
   - 환율은 홈탭 위젯에서 비동기로 로딩

4) 기존 Onboarding 컴포넌트(App.jsx 내부):
   - 기존 코드 주석 처리, 새 OnboardingSimple 컴포넌트로 교체
   - 파일: src/components/OnboardingSimple.jsx

5) 디자인:
   - Toss 첫 실행 스타일: 여백 많고, 글씨 크고, 선택지 명확
   - 드롭다운: 바텀시트로 (아래에서 올라오는 선택 패널)
   - 전환 애니메이션: fade + slide up

### 안건 008: 통합검색 고도화

Board-002에서 만든 SearchTab.jsx를 고도화.

1) 중국어 검색 지원:
   - pocketCategories의 name.zh, description.zh도 검색 대상에 포함
   - recommendedCourses의 name.zh, tags도 포함
   - 예: "美食" 검색 -> 맛집 포켓 결과 표시

2) 검색 결과 카테고리 분류:
   - "서비스" 섹션: 매칭된 포켓들
   - "코스" 섹션: 매칭된 추천 코스들
   - "가이드" 섹션: 매칭된 가이드들 (입국카드, SIM, 세금환급, 면세한도)
   - 각 섹션 헤더 표시, 결과 없는 섹션은 숨김

3) 검색 결과 카드 터치 동작:
   - 포켓 결과 -> 서비스탭 해당 포켓으로 이동
   - 코스 결과 -> 코스탭 해당 코스 상세로 이동
   - 가이드 결과 -> 해당 가이드 전체화면으로 표시

4) 빈 상태 개선:
   - 검색 전: "무엇을 찾고 계세요?" + 인기 검색어 칩
   - 결과 없음: "검색 결과가 없습니다. 다른 키워드로 검색해보세요." + 추천 키워드 3개

5) 검색 UX:
   - 검색어 입력 시 실시간 필터링 (debounce 300ms)
   - 검색어 지우기 X 버튼
   - 최근 검색어 개별 삭제 + 전체 삭제

### 안건 009: 스켈레톤 로딩

모든 비동기 데이터 로딩 화면에 Toss 스타일 스켈레톤 적용.

1) 스켈레톤 컴포넌트 만들기:
   - src/components/common/Skeleton.jsx
   - props: width, height, rounded (boolean), className
   - 스타일: bg-gray-200 animate-pulse rounded-lg
   - variant: text (h-4), card (h-24 rounded-2xl), circle (rounded-full)

2) 적용 위치:
   - 홈탭 위젯 로딩 시 (날씨, 환율 API 호출 중)
   - 코스탭 지도 로딩 시
   - 검색 결과 로딩 시
   - TourAPI 데이터 로딩 시 (TourSpotSection)

3) 기존 LoadingSpinner.jsx:
   - 유지하되, 전체 페이지 로딩에만 사용
   - 부분 로딩은 Skeleton 사용

4) 스켈레톤 패턴:
   - 카드 스켈레톤: 제목줄(w-3/4 h-4) + 설명줄(w-full h-3) + 설명줄(w-2/3 h-3)
   - 위젯 스켈레톤: 아이콘 원형(w-8 h-8) + 숫자줄(w-1/2 h-6)
   - 리스트 스켈레톤: 3개 반복
