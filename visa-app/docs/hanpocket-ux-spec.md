## 0. 서비스 정체성

| 항목 | 내용 |
|------|------|
| 서비스명 | HanPocket (韩口袋) |
| 슬로건 | "계획의 50%를 버린 여행이 진짜 여행이다" |
| 타겟 | 한국에 오는 중국인 (관광객 / 유학생 / 거주자 / 사업자) |
| 언어 | 중국어(간체) 우선 → 한국어 → 영어 |

| 포지셔닝 | 구글맵=기능, 트립어드바이저=정보, **HanPocket=감성(여행의 동반자)** |
