## HanPocket 제품 철학 (범범뻠 원칙)
1. **사용성**: 대중 플랫폼(위챗/샤오홍슈/카카오) 수준으로 익숙하고 편해야 함
2. **이중 시장**: 중국에서도, 한국에서도 쓸 수 있어야 함
3. **트렌드**: 최근 3년 트렌드와 맞닿아 있어야 함 (유행 지난 건 OUT)
4. **유일성**: "이런 앱은 전세계에 없다" — 그만큼 차별화된 가치
5. **진심**: 1단계부터 중국인에게 진짜 도움이 되어야 함. 마음이 느껴져야 함
6. **수익**: 동시에 돈도 벌 수 있는 구조여야 함
- 진심 + 수익이 공존하는 서비스. 둘 중 하나만 있으면 안 됨.
