## 핵심 결정사항
- **홈탭 퀵슬롯 → 위젯 그리드로 교체** (위챗/알리페이 스타일, Claude Code 진행 중)
- **지도탭 → "코스 플래너"로 변신** (하단탭 이름: 코스/路线)
  - 바이두 = A→B 실행, HanPocket = "어디를 어떤 순서로" 계획
  - **추천 코스 = HanPocket의 차별성이자 아이덴티티** ← 범범뻠 강조
  - 기존 미니맵은 각 기능(병원/약국/화장실 등) 안에서 유지
  - 바이두 딥링크로 실제 길찾기 연동
