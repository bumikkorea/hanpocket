1. 쿠팡 파트너스

한국 최대 쇼핑 어필리에이트. HanPocket 쇼핑탭 핵심 수익원.

https://partners.coupang.com
쿠팡 파트너스 공식 사이트

가입 절차 (5분)

  1. 1위 링크 접속 → "회원가입"
  2. 2쿠팡 계정으로 로그인 (없으면 쿠팡 회원가입 먼저)
  3. 3파트너스 가입 정보 입력:
    • 미디어 유형: "웹사이트/블로그" 선택
    • URL: https://hanpocket.pages.dev
    • 카테고리: "라이프스타일"
    • 월 방문자: "1,000 미만" (처음이니까)
  4. 4정산 정보 입력: 본인 계좌 + 주민등록번호 (세금 신고용)
  5. 5가입 완료 → 즉시 승인 (심사 없음!)

링크 생성법

  1. 대시보드 → "링크 생성"
  2. 상품 URL 붙여넣기 or 검색
  3. 생성된 어필리에이트 URL 복사 → HanPocket에 삽입
수수료: 전 상품 3% (로켓배송 3.5%) / 최소 정산: 10,000원 / 매월 25일 지급
팁: "딥링크" 기능으로 어떤 쿠팡 페이지든 어필리에이트 링크로 변환 가능. 카테고리 페이지, 기획전 페이지도 OK.

2. 클룩 (Klook) 어필리에이트

여행/액티비티 어필리에이트. HanPocket 여행탭 수익원.

https://affiliate.klook.com
Klook Affiliate Program

가입 절차 (5분)

  1. 1위 링크 접속 → "Join Now" or "가입하기"
  2. 2계정 생성 (이메일 + 비밀번호)
  3. 3사이트 정보 입력:
    • Website URL: https://hanpocket.pages.dev
    • Category: "Travel"
    • Monthly Traffic: "Under 5,000"
    • Description: "Korean lifestyle app for Chinese tourists and residents"
  4. 4제출 → 1~3일 내 이메일로 승인

링크 생성법

  1. 승인 후 대시보드 로그인
  2. "Links & Tools" → 상품 검색 or URL 입력
  3. 생성된 링크 복사 → HanPocket 여행탭에 삽입
수수료: 예약 완료 시 2~5% / PayPal or 은행 송금 / 매월 지급
팁: 한국 내 액티비티 (한복체험, 테마파크, 투어 등)가 특히 잘 맞음. 중국어 페이지 자동 지원!

https://www.linkprice.com
링크프라이스 — 국내 1위 CPS 어필리에이트 네트워크

  2. 2회원 정보 입력:
    • 매체 유형: "웹사이트"
    • 매체 URL: https://hanpocket.pages.dev
    • 매체 설명: "재한 중국인 대상 한국 생활 슈퍼앱"
  3. 3가입 후 → 개별 광고주에게 "제휴 신청"
  4. 4광고주가 승인하면 링크 생성 가능 (보통 1~3일)

4. 애드픽 (Adpick)

CPA 네트워크. 앱 설치/회원가입 건당 수수료. 빠른 수익 가능.

https://www.adpick.co.kr
애드픽 — CPA 성과형 어필리에이트

가입 절차 (3분)

  1. 1위 링크 접속 → "회원가입"
  2. 2SNS 계정 or 이메일로 가입
  3. 3프로필에 활동 채널 등록: URL 입력
  4. 4즉시 승인 → 바로 캠페인 참여 가능

수익 구조

  • 앱 설치: 건당 100~500원
  • 회원가입: 건당 500~2,000원
  • 상담신청/견적: 건당 1,000~5,000원
  • 정산: 5,000원 이상이면 즉시 출금 가능
팁: 금융(카드 발급), 보험, 통신사 캠페인이 단가 높음. 재한 중국인에게 필요한 서비스와 매칭하면 전환율 좋음.

블로그 콘텐츠 + 쇼핑 연계 수익. 장기적 SEO 효과.

https://in.naver.com
네이버 인플루언서 센터

가입 절차 (10분)

  1. 1위 링크 접속 → 네이버 로그인
  2. 2"인플루언서 지원" 클릭
  3. 3분야 선택: "여행" or "리빙"
  4. 4활동 채널 등록:
    • 네이버 블로그 (있으면)
    • 외부 채널: https://hanpocket.pages.dev
  5. 5심사 기간: 1~2주 (콘텐츠 품질 기반)
주의: 네이버 블로그에 최소 20~30개 포스팅이 있어야 승인 확률 높음. 블로그가 없으면 나중에 신청해도 됨.
수익 구조: 인플루언서 검색 노출 → CPC 광고 수익 + 쇼핑 커미션

8. 아마존 어소시에이트 (선택)

글로벌 상품 연결. 한국에서 직배송 가능 상품 추천 시 유용.

https://affiliate-program.amazon.com
Amazon Associates — 글로벌 어필리에이트

가입 절차 (10분)

  1. 1위 링크 접속 → "Sign Up"
  2. 2아마존 계정으로 로그인 (없으면 생성)
  3. 3사이트 등록: URL + 설명 입력
  4. 4정산 방법: 은행 계좌 직접 입금 or 아마존 기프트카드
  5. 5가입 후 180일 내 3건 이상 판매 필요 (안 하면 해지)
주의: 180일 내 3건 판매 조건. HanPocket 초기엔 후순위로 하고, 트래픽 올라오면 가입 추천.
수수료: 카테고리별 1~10% (뷰티 6%, 패션 7%, 전자 3%)
