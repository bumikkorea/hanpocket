# HEARTBEAT.md — 조건부 자율 작업 스케줄러

## 🎯 **조건부 하트비트 시스템** (Claude 사용량 최적화)

### 하트비트 작동 조건:
1. **TASKQUEUE.md에 미완료 작업(`[ ]`)이 있을 때만** 하트비트 실행
2. **모든 작업이 완료([x])되면** 하트비트 자동 중단 
3. **새 작업이 추가되면** 하트비트 자동 재시작

### 매 하트비트 시 수행:
1. `TASKQUEUE.md` 읽기
2. 미완료(`[ ]`) 작업 확인
   - **있으면**: 첫 번째 작업 실행
   - **없으면**: **"🎯 모든 자율작업 완료! 하트비트 중단."** 알림 후 대기모드
3. 실행 → 완료 체크 + 로그 기록
4. 범범뻠 알림 규칙 적용

### 알림 규칙:
- **작업 완료** → 조용히 로그만 (알림 X)
- **모든 작업 완료** → **"🎯 완료! 하트비트 중단"** 1회 알림
- **에러/실패** → 범범뻠에게 알림
- **결정 필요** → 범범뻠에게 질문  
- **큰 마일스톤** → 간단히 보고

### 💰 사용량 최적화:
- **불필요한 반복 체크 방지**
- **Claude Max 구독 사용량 절약**
- **필요할 때만 정확히 작동**

### 재시작 방법:
- **TASKQUEUE.md에 새 `[ ]` 작업 추가** → 자동 감지 후 하트비트 재시작
- **"작업 확인해줘"** 수동 명령어로 즉시 체크 가능
