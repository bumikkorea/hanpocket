# 범범뻠 수동 팩트체크 리스트

## 일회성 확인 (론칭 전 1회)
- [ ] 입국카드 양식 — 현재 양식 변경 여부 (출입국관리사무소)
- [ ] 입국카드 작성 항목 수 (14개? 15개?)
- [ ] eSIM 추천 업체 실제 존재 여부 + URL 정상 여부 (유심사, 에어알로, 도시락)
- [ ] 인천공항 T1/T2 SIM 판매 부스 위치 정확성
- [ ] 인천공항 세금환급 카운터 위치 (T1: 28번 게이트 근처, T2: 253번 게이트 근처)
- [ ] 바이두맵 딥링크 한국 내 작동 여부 (한국 귀국 후 테스트)

## 반기 업데이트 (매년 1월, 7월) 🔄
- [ ] 세금 환급 최소 구매금액 — 현재 15,000원 맞는지? (국세청/관세청)
- [ ] 세금 환급 방법 3가지(즉시/시내/공항) 변경 여부
- [ ] 중국 입국 면세 한도 — 현재 5,000위안 맞는지? (중국 해관총서)
- [ ] 주류 반입 한도 — 1병 1.5L 이하 맞는지?
- [ ] 담배 반입 한도 — 400개비(2보루) 맞는지?
- [ ] 향수 반입 규정

## 분기 업데이트 (매 3개월) 🔄🔄
- [ ] eSIM 가격대 변동 확인 (유심사/에어알로/도시락)
- [ ] 공항 SIM카드 가격대 변동 확인
- [ ] 포켓와이파이 가격대 변동 확인
- [ ] 세금 환급 키오스크 위치 변동 확인

## 연간 업데이트 (매년 1월) 🔄🔄🔄
- [ ] 액체류 기내 반입 규정 (100ml/투명지퍼백/1L) 변경 여부
- [ ] 입국카드 양식 변경 여부
- [ ] 추천 코스 10개 장소 영업 여부 확인
- [ ] 중국 해관총서 규정 전면 개정 여부

## 크로스체크 불가 항목 (AI 검증 미완)
- 모든 세금/면세 수치: 국세청/관세청 원천 접근 불가로 web_fetch 실패
- 해결: 범범뻠 브라우저에서 직접 확인 후 체크
