# HanPocket 과금 체계 확장 아이디어

## 현재 등급제 (기준)

- 🟢 **초록 (무과금)**: 주머니 전체, 긴급 전체, 한국어 1~3급, 글쓰기 X
- 🔵 **파란 (여행객)**: 무료 — 위 + 글쓰기 2개/일
- 🔴 **빨간 (유학생)**: 유료 — 위 + 한국어 4~6급, 글쓰기 무제한
- ⚫ **검정 (거주/영주)**: 유료 — 위와 동일
- 🌈 **무지개 (사업자)**: 유료 — 위 + 홍보글 작성 가능

## 1. 프리미엄 기능 아이디어

### 📱 AI 기반 서비스
- **AI 번역 횟수 제한**
  - 무료: 10회/일
  - 유료: 무제한 + 고품질 번역 모델
  - 프리미엄: 실시간 음성 번역 + OCR 번역

- **AI 맞춤 추천**
  - 개인화된 맛집/관광지 추천
  - 비자 상담 AI 챗봇 (고급 모델)
  - 한국 생활 스케줄 자동 생성

### 📝 콘텐츠 & 커뮤니티
- **다이어리 기능**
  - 무료: 로컬 저장만
  - 유료: 클라우드 동기화
  - 프리미엄: 다이어리 SNS 공유 + 추억 앨범 생성

- **프리미엄 콘텐츠**
  - 한국어 고급 과정 (7~10급)
  - VIP 맛집 할인 쿠폰
  - 독점 한류 콘텐츠 정보

### 🎯 편의 서비스
- **맛집 예약 대행**
  - 기본: 예약 링크 제공
  - 프리미엄: 대행 예약 서비스 (수수료 포함)
  - VIP: 인기 맛집 우선 예약권

- **개인 비서 서비스**
  - 병원 예약 대행
  - 관공서 업무 안내
  - 은행/통신사 업무 번역 지원

## 3. 광고 모델

### 배너 광고
- **무료 사용자**: 하단 배너 상시 노출
- **유료 사용자**: 배너 제거 혜택

### 네이티브 광고
- **맞춤형 콘텐츠**: 맛집/쇼핑 추천에 자연스럽게 삽입
- **지역별 타겟팅**: GPS 기반 주변 업체 광고

### 스폰서드 콘텐츠
- **특집 기사**: "한국 생활 꿀팁" 내 브랜드 콘텐츠
- **이벤트 섹션**: 브랜드 협찬 이벤트 노출

### 광고 수익 분배
- **무료 사용자**: 100% 광고 수익
- **유료 사용자**: 프리미엄 기능으로 수익 보완
- **CPM 예상**: ¥5-15 (한국 거주 중국인 타겟)

## 4. 어필리에이트 수익 모델

### 🛍️ 쇼핑 제휴
- **온라인 몰**: 쿠팡, G마켓, 11번가 (3-7% 수수료)
- **면세점**: 롯데, 신라, 동화 (5-10% 수수료)
- **한국 브랜드**: 코스메틱, 패션 브랜드 직접 제휴

### 🍽️ 맛집 & 배달
- **배달의민족/요기요**: 주문 수수료 (5-8%)
- **맛집 예약**: 키친 딜라이트, 테이블링 (예약당 ¥5-20)
- **프랜차이즈**: BBQ, 교촌치킨 등 (할인 쿠폰 제공 대가)

### 🏠 생활 서비스
- **부동산**: 직방, 다방 (계약 성사 시 1-2% 수수료)
- **통신**: SK, KT, LG (가입당 ¥50-100)
- **은행**: 계좌 개설, 신용카드 발급 (건당 ¥30-80)

### 📚 교육 & 자격증
- **한국어 학원**: 가나다, YBM 등 (수강료의 10-15%)
- **자격증**: 컴활, 토픽 등 인강 플랫폼 제휴
- **대학교**: 어학당 등록 시 리베이트

## 5. B2B 모델

### 🏛️ 정부/공공기관 제휴
- **한국관광공사**: 관광 콘텐츠 제작, 프로모션 (연 ¥500만-1,000만)
- **법무부 출입국 관리소**: 비자 정보 제공 플랫폼 (연 ¥200만-500만)
- **지자체**: 지역 홍보 콘텐츠 (건당 ¥10만-50만)

### 🏪 기업 제휴
- **면세점**: 롯데, 신라, 현대 (독점 할인 혜택 + 수수료)
- **호텔 체인**: 롯데, 신라, 메리어트 (예약 수수료 + 프로모션)
- **항공사**: 대한항공, 아시아나 (마일리지 제휴, 항공권 할인)

### 🏢 스타트업 & 중소기업
- **한국 진출 중국 기업**: 앱 내 브랜드 홍보 공간 제공
- **배송 대행업**: 직구 서비스 제휴 (거래액의 2-3%)
- **번역/통역 업체**: 전문 서비스 연결 (중개 수수료 15-20%)

## 6. 결제 연동 방안

### 🇨🇳 중국 결제 수단

- **알리페이 (Alipay)**: 보완재 — 해외결제 강점
- **은련카드 (UnionPay)**: 카드 결제 선호층

### 🇰🇷 한국 결제 수단  
- **카카오페이**: 한국 거주자 편의성
- **네이버페이**: 쇼핑 제휴 시너지
- **토스페이**: 간편 결제 + 환전 서비스

### 💳 글로벌 결제
- **페이팔 (PayPal)**: 다국적 사용자
- **애플페이/구글페이**: 앱스토어 연동
- **VISA/마스터카드**: 백업 수단

### 🔄 환전 & 수수료
- **실시간 환율**: API 연동으로 투명한 가격 표시
- **결제 수수료**: 2.5-3.5% (결제 대행사 + 앱 수수료)
- **환불 정책**: 7일 무료 취소, 14일 부분 환불
