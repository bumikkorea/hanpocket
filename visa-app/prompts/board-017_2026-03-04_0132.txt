# Board-017: API 실데이터 연동 + 기존 데이터 연결 + 음성재생

## 목표
Mock 데이터를 실 API 데이터로 교체하고, 이미 있는 데이터 파일을 포켓에 연결하고, 회화 포켓에 음성재생 추가.

## 1. TourAPI 연동

### API 기본 설정
- Base URL: `https://apis.data.go.kr/B551011`
- 서비스키: `40490c648d17e72faabd00bfc8be9994ee2cc64946edb68f4beb374eae1d482b`
- 중국어: `ChsService2`, 한국어 폴백: `KorService2`
- 공통 파라미터: `MobileOS=ETC&MobileApp=HanPocket&_type=json&numOfRows=20`

### 1b. 관광지 데이터 연동 (여행 포켓용)
API: `GET /ChsService2/areaBasedList2?serviceKey=...&areaCode=1&contentTypeId=12`
- areaCode: 1=서울, 6=부산, 39=제주
- contentTypeId: 12=관광지, 14=문화시설, 15=축제, 25=여행코스

## 2. 기존 데이터 파일 → 포켓 연결

### 2a. 의료 포켓 강화
- `src/data/hospitalData.js`와 `src/data/seoulHospitals.js`가 이미 있음

  - 검색 기능
  - 카카오맵 딥링크
  - 긴급 연락처 (119, 1339)

## 3. 날씨 비교 실시간화

- API: `https://wttr.in/Seoul?format=j1` (무료, 키 불필요)
- `https://wttr.in/Beijing?format=j1`
- 새 파일 또는 인라인: fetch → 온도/날씨 아이콘 추출 → 렌더링
- 에러 시 mock 데이터 폴백

## 4. 회화 포켓 음성재생 추가
- `KoreanPhraseCard.jsx` (공통 컴포넌트, 300줄)에 이미 `speak` 함수가 있는지 확인
- 있으면: 각 회화 카드의 한국어 텍스트 옆에 🔊 버튼 추가 → Web Speech API `speechSynthesis.speak()` 호출
- 없으면: `window.speechSynthesis`를 사용하는 speak 함수 추가
  ```js
  function speakKorean(text) {
    const u = new SpeechSynthesisUtterance(text);
    u.lang = 'ko-KR';
    u.rate = 0.8;
    speechSynthesis.speak(u);
  }
  ```
- 대상: RestaurantPocket, CafePocket, TransportPocket, ConveniencePocket, AccommodationPocket, EmergencyPocket
