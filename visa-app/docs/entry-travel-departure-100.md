# HanPocket 입국준비/여행중/출국준비 — 100가지 편의·공감 포인트

> 타겟: 중국, 태국, 인도네시아, 말레이시아, 베트남, 필리핀 등 동남아 + 동아시아 여행객
> 핵심: "이 앱이 나를 이해하고 있다"는 느낌 → 편의 + 정성 + 공감

## 🛬 A. 출국 전 / 입국준비 (출발국에서)

### 🇨🇳 중국 특화
1. **VPN 알림** — "한국에서는 VPN 없이 구글/유튜브/인스타 됩니다!" (중국인에게 가장 놀라운 점)
2. **위챗페이/알리페이 한국 사용 가이드** — 어디서 되고 어디서 안 되는지 (명동 OK, 시장 X)
3. **중국 운전면허 → 한국 국제면허** — 한국에서 렌트카 가능 여부 (불가, 제주 제외)
4. **약 반입 주의** — 중국 감기약(含에페드린) 한국 반입 금지 품목 안내
5. **중국 은행카드(银联) 한국 ATM 사용** — 어디서 인출 가능? 수수료는?
6. **가족 단톡방용 템플릿** — "엄마 나 한국 도착했어!" 위챗 메시지 템플릿 (중국 부모님 안심용)
7. **한국 전화번호 없이 가능한 것들** — 카카오 가입, 네이버맵 등 비회원 기능 안내
8. **날씨 비교 카드** — "지금 베이징 -5°C / 서울 -2°C" 출발 전 옷차림 가이드
9. **중국 공항 면세점 vs 한국 면세점 가격 비교** — 어디서 사는 게 이득?
10. **비행기 내 입국카드 중국어 번역** — 실물 카드 각 항목 1:1 번역

### 🇹🇭 태국 특화
11. **사원 방문 복장 안내** — 한국 사찰은 복장 제한 없음 (태국과 다름!)
12. **왕실 모독법 → 한국은 없음** — 한국에서 정치 이야기 자유롭게 가능
13. **매운맛 레벨 비교** — "한국 매운맛 vs 태국 매운맛" 재미 카드
14. **태국 바트 → 원화 환전** — 태국 내 환전 vs 한국 도착 후 환전 어디가 유리
15. **불교 문화 공통점** — 한국 사찰 템플스테이 추천 (태국인 공감)
16. **선풍기/에어컨 문화 차이** — 한국 겨울엔 온돌 (바닥 난방) 설명
17. **툭툭 → 택시앱** — 카카오T 설치 안내 (태국 Grab 같은 것)

### 🇮🇩 인도네시아 특화
18. **할랄 음식 가이드** — 한국 할랄 식당 위치 + 무슬림 프렌들리 구역
19. **기도실(Mushola) 위치** — 서울 주요 관광지 근처 이슬람 예배실 안내
20. **금요 예배(Jumat) 모스크** — 서울중앙성원(이태원) + 기타 기도 장소
21. **돼지고기 주의** — 한국 음식 대부분에 돼지고기/돼지육수 포함, 확인법 안내
22. **비데/물 세정** — 한국 화장실 비데 사용법 (인도네시아 문화와 유사)
23. **인도네시아 루피아 환전** — 소액권이 많아서 환전 시 주의사항
24. **라마단 기간 한국 여행** — 해 뜨는/지는 시간 알림, 이프타르 가능 식당
25. **인도네시아어 긴급 카드** — "저는 무슬림입니다. 돼지고기/술 안 됩니다" 한국어 카드

### 🇲🇾 말레이시아 특화
26. **다민족 고려** — 말레이계(무슬림)/중국계(화교)/인도계 각각 다른 식사 제한
27. **화교 말레이시안** — 중국어 가능! 중국어 가이드 바로 연결
28. **말레이시아 영어** — 한국에서 영어 통하는 곳/안 통하는 곳 안내
29. **듀리안 반입 금지** — 한국행 비행기에 듀리안 못 가져감!
30. **할랄 인증 마크** — 한국 할랄 인증 시스템 설명 (말레이시아 JAKIM과 다름)

### 🇻🇳 베트남 특화
31. **베트남 동 → 원화** — 환전 시 0 세기 주의 (10만 동 ≈ 5,500원)
32. **쌀국수 vs 한국 라면** — 한국 편의점 쌀국수 옵션 안내
33. **오토바이 문화 → 한국은 도보+지하철** — 교통 패러다임 완전히 다름
34. **베트남 커피 vs 한국 카페** — 한국 카페 문화 소개

### 🇵🇭 필리핀 특화
35. **타갈로그어 긴급 카드** — 기본 한국어 표현 + 긴급 연락처
36. **카톨릭 문화 공통점** — 한국 명동성당 등 성당 안내
37. **필리핀 페소 환전** — 한국 내 환전 가능 여부 (일부 은행만)
38. **영어 소통** — 한국에서 영어 통하는 관광지 리스트

## 🧒👨‍👩‍👧‍👦 B. 동반자별 특화

### 유아 동반 (0-6세)
39. **기내 유아식 사전 예약** — 항공사별 유아식 메뉴 안내
40. **한국 공항 수유실/기저귀교환대 위치** — 인천공항 T1/T2 상세 지도
41. **유모차 대여** — 인천공항 + 서울 주요 관광지 유모차 렌탈
42. **한국 분유/기저귀 브랜드** — 현지에서 구매 가능한 제품 (하기스, 보솜이)
43. **소아과 긴급** — 외국인 소아 진료 가능 병원 리스트
44. **이유식 식당** — 아기 의자 + 이유식 데워주는 식당 표시
45. **놀이터/키즈카페 위치** — 가까운 실내 놀이 공간
46. **한국 택시 카시트** — 카카오T 카시트 옵션 유무
47. **아이 미아 방지** — 한국어 이름표 카드 생성 (이름+부모연락처+호텔)
48. **기저귀 쓰레기봉투** — 한국 편의점에서 비닐봉투 구매법

### 어르신 동반 (60+)
49. **휠체어/보행기 공항 서비스** — 인천공항 휠체어 무료 서비스 예약
50. **에스컬레이터/엘리베이터 위치** — 지하철역별 배리어프리 정보
51. **노인 우대** — 지하철 65세 이상 무료 (외국인도!)
52. **느린 식당** — 천천히 식사 가능한 조용한 식당 추천
53. **약국 필수 약품** — 고혈압/당뇨약 한국어 표기 + 동일 성분 약 안내
54. **온돌방 추천** — 관절에 좋은 바닥 난방 숙소 추천
55. **경로석 문화** — 한국 지하철 경로석은 진짜 비워둠 (문화 설명)
56. **응급 상황 카드** — "이 분은 ○○ 질환이 있습니다" 한국어 의료 카드
57. **대형 글씨 모드** — 앱 내 시니어 모드 (큰 글씨 + 단순 UI)

### 커플/신혼부부
58. **커플 사진 스팟** — 인스타/샤오홍슈 인기 촬영 장소 TOP 20
59. **웨딩 촬영** — 한국 스튜디오 웨딩 촬영 패키지 (중국인 인기)
60. **커플링 쇼핑** — 명동/홍대 커플링 전문점
61. **로맨틱 디너** — 한강뷰/야경 레스토랑 예약 가이드
62. **커플 한복 체험** — 경복궁 앞 한복 대여 + 무료 입장

### 친구 그룹 (3-6인)
63. **단체 택시 vs 대형택시** — 4인 이상이면 대형택시(JUMBO) 호출
64. **단체 식사** — 인원수별 추천 메뉴 (삼겹살 4인, 해물탕 6인)
65. **단체 숙소** — 에어비앤비 vs 게스트하우스 비교
66. **더치페이 문화** — 한국은 보통 1인이 계산 (나눠내기 어색할 수 있음)

## ✈️ C. 비행편 연동 / 실시간 추적 (출국→입국 여정)

### 비행편 예약 정보 연동
67. **예약 확인번호(PNR) 입력** → 항공편 정보 자동 로드
68. **탑승권 QR 스캔** → 항공편 + 좌석 + 게이트 정보 파싱
69. **출발 공항 가이드** — 출발 공항(PVG, BKK, CGK 등) 체크인 카운터 위치
70. **비행 시간 카운트다운** — "한국까지 2시간 43분!" 실시간 표시
71. **시차 알림** — "한국은 지금 오후 3시예요. 도착하면 저녁이에요!"
72. **실시간 항공편 위치** — 지도 위에 비행기 아이콘 (FlightRadar24/AviationStack API)
73. **기내 한국어 연습** — 비행 중 할 수 있는 10분 한국어 미니 레슨
74. **도착 날씨 미리보기** — "서울 도착 시 기온 8°C, 맑음 ☀️"
75. **인천공항 도착 후 동선** — 입국심사 → 수하물 → 세관 → 교통 경로 시각화
76. **수하물 벨트 번호** — 항공편별 수하물 컨베이어 벨트 (인천공항 API)
77. **입국 대기 시간** — 현재 입국심사 예상 대기 시간

### 감성 터치 (한국 오고 있다는 실감)
78. **"한국이 기다리고 있어요"** — 비행 경로 위 한국 아이콘 점점 커짐
79. **한국 진입 알림** — GPS 기반 "한국 영공에 진입했습니다! 🇰🇷"
80. **도착 공항 라이브캠** — 인천공항 실시간 웹캠 영상
81. **환영 메시지** — 착륙 감지 시 "환영합니다! Welcome to Korea! 🎉"
82. **현지 시간 자동 전환** — "시계를 한국 시간으로 바꿨어요"
83. **첫 한국어 한마디** — "감사합니다" 발음 가이드 (착륙 후 팝업)

## 🗼 D. 여행 중 보강 데이터

### 식사/음식
84. **채식/비건 식당 필터** — 한국은 채식 옵션 매우 적음, 전용 가이드 필수
85. **알레르기 카드** — "저는 ○○ 알레르기가 있습니다" 한국어 카드 생성
86. **음식 사진 → 메뉴 번역** — 카메라로 메뉴판 찍으면 자동 번역
87. **한국 식사 에티켓** — 젓가락 밥에 꽂지 마세요, 어른 먼저 수저
88. **편의점 할랄/비건 표시** — CU/GS25 할랄 표시 제품 리스트
89. **물 문화** — 한국 수돗물 음용 가능, 식당 물 무료

### 교통
90. **지하철 임산부석** — 핑크 좌석 = 임산부 전용, 앉지 마세요
91. **버스 하차 벨** — 내릴 때 미리 벨 누르기 (동남아에는 없는 문화)
92. **택시 팁 문화** — 한국은 팁 없음! (동남아와 다름)
93. **자전거 대여** — 따릉이(서울) 외국인 이용법

### 쇼핑
94. **사이즈 변환표** — 중국/태국/한국 의류 사이즈 대조표
95. **교환/환불 문화** — 한국은 7일 이내 교환 가능 (영수증 필수)
96. **시장 흥정** — 남대문/동대문은 흥정 가능, 올리브영은 정찰제

### 안전/문화
97. **야간 안전** — "한국은 새벽 2시에도 편의점 혼자 가도 안전합니다"
98. **음주 문화** — 소주 도수 17%, 무슬림이면 "안 마셔요" 카드
99. **사진 촬영 매너** — 한국인 얼굴 무단 촬영 주의, 군사시설 촬영 금지

## 🔌 F. 비행편 실시간 추적 — 기술 구현

### 무료/저가 API 옵션
| API | 무료 티어 | 실시간 | 비고 |
|-----|----------|--------|------|
| **AviationStack** | 100회/월 | ✅ | 비행편 상태, 공항 데이터 |
| **FlightAware AeroAPI** | 제한적 | ✅ | 가장 정확, 유료 |
| **OpenSky Network** | 무제한 | ✅ | 오픈소스, ADS-B 기반 |
| **ADS-B Exchange** | 무제한 | ✅ | 커뮤니티 기반, 무료 |

### 추천 구현
1. **Phase 1**: 항공편 번호 입력 → AviationStack으로 출발/도착/상태 표시
2. **Phase 2**: OpenSky로 실시간 위치 → 지도 위 비행기 아이콘
3. **Phase 3**: 푸시 알림 (출발/착륙/지연/게이트변경)

### 사용자 플로우
```
예약 확인번호 입력
    ↓
항공편 자동 매칭 (CZ3066, TG658 등)
    ↓
출발 24시간 전: "내일 한국 갑니다! ✈️" 알림
    ↓
출발 3시간 전: 공항 체크인 안내 + 게이트 정보
    ↓
이륙: "이륙했습니다! 한국까지 3시간 12분"
    ↓
비행 중: 실시간 지도 + 한국어 미니레슨 + 도착 날씨
    ↓
한국 영공 진입: "곧 도착합니다! 🇰🇷"
    ↓
착륙: "환영합니다!" + 입국 동선 가이드 자동 시작
```
