HanPocket 개발자 API 가입 가이드

범범뻠용 — 네이버 + data.go.kr + DeepSeek + law.go.kr 가입 총정리

1. 네이버 개발자 센터

네이버 로그인 (네아로) + 뉴스/검색 API를 한 곳에서 발급받을 수 있음

https://developers.naver.com

가입 & 앱 등록 (5분)

  1. 1위 링크 접속 → 우측 상단 "로그인" (기존 네이버 계정으로)
  2. 2로그인 후 상단 메뉴 "Application" → "애플리케이션 등록" 클릭
  3. 3앱 정보 입력:
    • 애플리케이션 이름: HanPocket
    • 사용 API 선택: 네아로 (네이버 로그인) 체크
    • 추가 선택: 검색 체크 (뉴스 검색용)
  4. 4네아로 환경 설정:
    • 환경 추가: PC웹 선택
    • 서비스 URL: https://hanpocket.pages.dev
    • Callback URL: https://hanpocket.pages.dev/
  5. 5"등록하기" 클릭 → 완료!

발급된 키 확인

"내 애플리케이션" → HanPocket 클릭 → 아래 2개 복사해서 구월에게 전달:

항목설명
Client ID공개 키 (프론트엔드에서 사용)
Client Secret비밀 키 (서버에서만 사용)
네이버 로그인과 뉴스 검색 API가 같은 Client ID/Secret으로 동작함. 한 번 등록이면 끝!

2. 공공데이터포털 (data.go.kr)

병원, 관광, 공연, 날씨, 대기오염 등 15개 공공 API 무료 사용

https://www.data.go.kr

가입 (3분)

  1. 1위 링크 접속 → "회원가입"
  2. 2일반회원 선택 → 본인인증 (휴대폰 or 공인인증서)
  3. 3회원정보 입력 후 가입 완료

API 키 발급 (API당 2분)

  1. 1로그인 후 검색창에 API 이름 검색 (아래 표 참고)
  2. 2해당 API 페이지에서 "활용신청" 클릭
  3. 3활용 목적: "모바일 앱 서비스 개발"
  4. 4신청 후 즉시~1시간 내 자동 승인 (일부 수동 승인 1~3일)

우선 발급할 5개 API

#검색어용도
1건강보험심사평가원 병원정보의료탭 병원 검색
2한국관광공사 관광정보여행탭 관광지
3공연전시 정보 (KOPIS)한류탭 공연/전시
4기상청 단기예보홈 날씨 위젯
5에어코리아 대기오염정보홈 미세먼지
발급된 "일반 인증키 (Encoding)" 를 각각 복사해서 구월에게 전달하면 됨. 5개 다 하는데 10분이면 충분!

3. DeepSeek API (번역 엔진)

HanPocket 실시간 번역 기능의 핵심. 월 ~5,000원

https://platform.deepseek.com

가입 & 키 발급 (3분)

  1. 1위 링크 접속 → "Sign Up" (구글 계정으로 가능)
  2. 2로그인 후 좌측 메뉴 "API Keys" 클릭
  3. 3"Create new API key" → 이름: hanpocket
  4. 4생성된 키를 바로 복사 (다시 볼 수 없음!)
  5. 5좌측 "Billing" → "Top Up" → $5 충전 (약 ₩7,000)
주의: API 키는 생성 직후에만 전체 표시됨. 반드시 바로 복사해서 구월에게 전달할 것!
$5 충전하면 약 2~3개월 사용 가능 (번역 API는 매우 저렴). 자동 충전 설정도 가능.

4. 법령정보 API (law.go.kr)

비자 관련 최신 법령 자동 업데이트용

https://open.law.go.kr

가입 & 키 발급 (5분)

  1. 1위 링크 접속 → "회원가입"
  2. 2본인인증 후 회원가입 완료
  3. 3로그인 → "오픈API 이용신청"
  4. 4신청서 작성:
    • 서비스명: HanPocket
    • 활용 목적: 외국인 비자/체류 관련 법령 정보 제공
  5. 5신청 후 1~3일 후 승인 (이메일 통보)
승인까지 시간이 걸리니 나머지와 함께 미리 신청해두면 좋음!

체크리스트

  • 네이버 개발자 센터 가입 + 앱 등록 (Client ID / Secret 복사)
  • data.go.kr 가입 + 5개 API 활용신청 (인증키 5개 복사)
  • DeepSeek 가입 + $5 충전 + API 키 생성 (키 복사)
  • law.go.kr 가입 + API 이용신청
  • 위 키들 전부 구월에게 전달
전체 소요시간: 약 25~30분
비용: DeepSeek $5 (₩7,000) — 나머지 전부 무료
