## 관리자 포켓 토글 시스템

### 배경
모든 포켓(서비스)에 대해 관리자(범범뻠)가 표시/숨김을 개별 제어할 수 있는 토글 시스템이 필요하다.
현재 미구현 포켓이 숨겨져 있어서 새로 추가된 콘텐츠(가이드 등)도 볼 수 없는 문제가 있다.

### 요구사항

#### 1. 관리자 패널 (프로필탭 내부)
- 프로필탭(나) 하단에 "관리자 설정" 섹션 추가
- 진입 조건: 숨겨진 트리거 (예: 프로필 탭에서 로고를 5번 터치하면 관리자 모드 활성화)
- 활성화 상태는 localStorage에 저장 (admin_mode: true/false)

#### 2. 관리자 패널 UI
- "포켓 표시 관리" 화면
- pocketCategories의 모든 카테고리 > 모든 포켓을 리스트로 표시
- 각 포켓 옆에 토글 스위치 (on/off)
- on = 서비스탭에서 유저에게 보임
- off = 서비스탭에서 숨김
- 기본값: IMPLEMENTED_POCKETS에 있는 것만 on, 나머지 off
- 토글 상태는 localStorage에 저장 (pocket_visibility: { pocketId: true/false })

#### 3. 서비스탭 연동
- 서비스탭(PocketsPage)에서 포켓 렌더링 시 pocket_visibility 체크
- visibility가 true인 포켓만 상단 활성 영역에 표시
- visibility가 false인 포켓은 "업데이트 중" 그룹에 표시 (기존 로직 유지)
- 관리자가 토글 on 하면 즉시 서비스탭에 나타남

#### 4. 토글 스위치 디자인
- Toss 스타일 토글: 둥근 슬라이더, on=검정 off=회색
- 포켓 아이콘 + 이름 + 토글 한 줄
- 카테고리별 구분선
- 상단에 "전체 켜기" / "전체 끄기" 버튼

#### 5. 추가 기능
- 토글 변경 시 하단에 토스트 메시지: "OO 포켓이 활성화되었습니다"
- 관리자 모드일 때 하단 탭 "나" 아이콘 옆에 작은 점 표시 (관리자 모드 활성 표시)

### 테스트
- 프로필 탭에서 로고 5번 터치 -> 관리자 설정 나타남
- 포켓 토글 on/off -> 서비스탭에서 즉시 반영
- 새로고침 후에도 토글 상태 유지 (localStorage)
- 관리자 모드 비활성화 -> 관리자 설정 숨김
