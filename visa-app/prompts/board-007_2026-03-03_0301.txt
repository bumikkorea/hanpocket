### 안건 010: 인천공항 실시간 여객편 검수

기존 FlightSearchCard.jsx와 airportApi.js 확인 및 보완.

1) 확인사항:
   - API 호출 정상 작동 (serviceKey, from_time, to_time 파라미터)
   - 도착/출발 탭 전환
   - 편명 검색 기능
   - 항공기 기종 표시 (aircraftData.js)
   - 에러 핸들링 (API 실패 시 안내 메시지)

2) 보완사항:
   - 결과 없을 때: "현재 시간대 운항편이 없습니다" 안내
   - API 키 만료/실패 시: "일시적으로 조회할 수 없습니다" + 인천공항 홈페이지 링크
   - 자동 새로고침: 5분마다 (캐시 정책 따름)
   - lang 파라미터: 중국어(C) 우선, 실패 시 한국어(K) 폴백

### 안건 011: 코스 플래너 시간표 스타일 고도화

CourseTab.jsx 코스 상세 화면을 시간표 스타일로 개선.

1) 코스 상세의 타임라인을 시간 블록으로:
   현재: 세로 타임라인 (경유지 나열)
   변경: 시간표 스타일 블록

   예시:
   09:00 [==== 경복궁 (2h) ====]
         -- 지하철 15min --
   11:15 [==== 북촌 (1h) ====]
         -- 도보 10min --
   12:25 [==== 인사동 점심 (1h) ====]

2) 구현:
   - 코스의 첫 경유지 시작시간을 09:00으로 기본 설정
   - 사용자가 시작시간 변경 가능 (터치하면 시간 선택)
   - 각 경유지 duration + transport duration 계산해서 자동 시간 배치
   - 블록 높이 = 체류시간에 비례 (1시간=60px)
   - 이동 구간: 점선 + 이동수단 아이콘 + 소요시간

3) "내 코스에 추가" 후:
   - 내 코스 목록에서도 시간표 미니 프리뷰 표시
   - 시작시간 수정 가능

4) 디자인:
   - 시간 라벨: 좌측 고정 (text-xs text-gray-400, w-12)
   - 블록: bg-gray-50 border-l-4 border-[#111827] rounded-r-xl p-3
   - 이동 구간: border-l-2 border-dashed border-gray-300

### 안건 013: 약국 데이터 연동

건강보험심사평가원 약국 API 연동 (이미 승인됨).

1) src/api/pharmacyApi.js 생성:
   - API: https://apis.data.go.kr/B551182/pharmacyInfoService/getParmacyBasisList
   - serviceKey: 기존 data.go.kr 키 사용
   - 파라미터: sidoCd(시도), sgguCd(시군구), numOfRows, pageNo
   - 응답: 약국명, 주소, 전화번호, 좌표(XPos, YPos)

2) 캐시: 약국 데이터는 하루 1회 갱신 (localStorage, 24시간 TTL)

3) UI: 서비스탭 medical 포켓 내부 또는 별도 pharmacy 포켓
   - 현재 위치 기반 근처 약국 리스트
   - 카드: 약국명 + 주소 + 전화 + 거리
   - 전화 버튼 (tel: 링크)
