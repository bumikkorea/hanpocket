## 관광객 킬러 콘텐츠 4종 일괄 제작

### 배경
중국인 관광객의 여행 타임라인(출발전 - 기내 - 도착 - 체류 - 출국) 순서에 맞춘 필수 가이드 4개를 만든다.
모두 Toss 스타일 카드형 원페이지 가이드. 탭 분리하지 말고 하나의 스크롤 페이지로.

### 공통 디자인 규칙
- Toss 스타일: 카드 기반, 여백 넉넉 (p-5), 화면당 1주제
- 29CM 기본 유지: Inter 폰트, 검정 단색, 흰 배경
- 각 가이드는 독립 컴포넌트로 만들기
- 3개 국어 (ko, zh, en)
- 카드 스타일: bg-white rounded-2xl p-5 border border-[#E5E7EB]
- 중요 수치는 크게 (text-2xl font-bold)
- 주의사항은 bg-red-50 border-red-200 카드로 강조
- 출처 표기 필수 (하단에 회색 작은 글씨)

### 변동성 데이터 최신화 정책 (필수 적용)

모든 가이드에 아래 규칙 적용:

1) 기준일자 표시: 각 가이드 상단에 회색 작은 텍스트로
   { ko: '정보 기준: 2026년 3월', zh: '信息基准: 2026年3月', en: 'As of March 2026' }

2) 데이터 등급별 처리:
   - A등급 (거의 안 바뀜: 입국카드 양식, 액체류 100ml, 면세 한도 구조): 구체적 수치 하드코딩 OK
   - B등급 (연 1~2회 변동: 환급 기준금액, 해관 규정): 하드코딩하되 "정책 변경될 수 있음" 면책 추가
   - C등급 (수시 변동: eSIM/SIM 가격, 통신사 요금): 구체적 가격 대신 가격대(range)만 표기 + 공식 사이트 외부 링크 연결

3) C등급 가격 표기 형식:
   잘못된 예: "유심사 5일 9,900원"
   올바른 예: "유심사 5일 약 1만원대" + [최신 가격 확인] 버튼 (공식 사이트 링크)
   링크 버튼 스타일: text-blue-600 underline text-sm

4) 가이드 하단 공통 푸터:
   { ko: '정보 기준: 2026년 3월 | 문의: hanpocket@email.com', zh: '信息基准: 2026年3月 | 联系: hanpocket@email.com', en: 'As of March 2026 | Contact: hanpocket@email.com' }
   스타일: text-xs text-gray-400 text-center mt-8

### 1. 입국카드 작성법 (src/components/guides/ArrivalCardGuide.jsx)

기내에서 작성하는 입국카드 가이드.

섹션 구성:
A) 헤더: "입국카드 작성법" / "入境卡填写指南" / "Arrival Card Guide"
B) 입국카드 이미지 (실제 카드 레이아웃을 CSS로 재현)
   - 각 칸에 번호 매기기 (1~15번)
   - 번호 터치하면 해당 칸 작성법 설명 표시
C) 칸별 작성 가이드:
   1. Family Name (성) - 여권 영문 성
   2. Given Name (이름) - 여권 영문 이름  
   3. Gender (성별) - Male/Female 체크
   4. Nationality (국적) - CHINESE
   5. Date of Birth (생년월일) - DD/MM/YYYY
   6. Passport No (여권번호)
   7. Home Address (본국 주소) - 영문 or 중문
   8. Purpose of Visit (방문목적) - Tourism 체크
   9. Flight No (편명) - 탑승 항공편
   10. Port of Boarding (출발지)
   11. Length of Stay (체류기간)
   12. Address in Korea (한국 숙소) - 호텔명+주소
   13. Phone (연락처)
   14. Signature (서명)
D) 팁 카드: "미리 호텔 영문주소 캡처해두세요!"
E) 출처: 법무부 출입국외국인정책본부

### 2. SIM/eSIM 구매 가이드 (src/components/guides/SimGuide.jsx)

도착 직후 통신 개통 가이드.

섹션 구성:
A) 헤더 + "인터넷 없으면 아무것도 못해요" 경고 카드
B) 3가지 옵션 비교 카드:

| 옵션 | 가격대 | 개통시간 | 추천 |
|------|--------|---------|------|
| eSIM (추천) | 5,000~15,000원 | 즉시 | 아이폰XS 이상 |
| 공항 SIM카드 | 20,000~55,000원 | 5분 | 모든 폰 |
| 포켓와이파이 | 3,000~5,000원/일 | 수령 5분 | 여러 기기 |

C) eSIM 추천 업체 (가격순):
   - 유심사 (yousimsa.com) - 5일 약 1만원대 [최신 가격 확인] -> https://www.yousimsa.com
   - 에어알로 (airalo.com) - 1GB 약 5천원대 [최신 가격 확인] -> https://www.airalo.com
   - 도시락eSIM - 무제한 3일 약 1만원대 [최신 가격 확인] -> https://www.dosirakmobile.com
   (가격은 가격대만 표기, C등급 데이터 정책 적용)
D) 공항 SIM 구매 위치:
   - 인천공항 T1: 1층 도착장 KT/SKT 부스
   - 인천공항 T2: 1층 도착장 KT 부스
   - 김포공항: 1층 도착장
E) 설정 방법: eSIM QR스캔 3단계 (아이폰/안드로이드 분리)
F) 주의 카드: "중국 폰번호 로밍 vs 한국 SIM 장단점"
G) 출처: 각 통신사 공식 사이트

주의: 가격은 2026년 기준 추정치로 표기하고, "실제 가격은 구매 시점에 확인하세요" 면책 문구 추가.

### 3. 세금 환급 가이드 (src/components/guides/TaxRefundGuide.jsx)

쇼핑 후 세금 환급 방법.

섹션 구성:
A) 헤더 + 큰 숫자 카드: "최대 환급률" 표시
B) 환급 조건 카드:
   - 외국인 여권 소지자
   - 1개 매장에서 1회 15,000원 이상 구매 (2025.7월 기준 하향, 종전 30,000원)
   - 구매일로부터 3개월 이내 출국
   - Tax Free 또는 Tax Refund 표시 매장에서 구매
C) 환급 방법 3가지:
   1. 즉시환급 (매장에서 바로): 구매 시 여권 제시, 세금 차감된 가격으로 구매
   2. 시내환급 (시내 환급 키오스크): 영수증+여권 가져가기
   3. 공항환급 (출국 시): 세관 도장 -> 환급 카운터
D) 환급 계산기 (간단한 인터랙티브):
   - 구매금액 입력 -> 예상 환급액 계산
   - 부가세 10% 기준, 실 환급률 약 7~8%
E) 공항 환급 위치:
   - 인천공항 T1: 출국장 28번 게이트 근처
   - 인천공항 T2: 출국장 253번 게이트 근처
F) 주의 카드: "영수증 잃어버리면 환급 불가!"
G) 출처: 국세청, 관세청

주의: 환급 기준금액은 정책 변경 가능, "최신 기준은 매장에서 확인하세요" 면책 추가.

### 4. 면세 한도 + 액체류 가이드 (src/components/guides/DutyFreeGuide.jsx)

출국 전 반드시 알아야 할 반입 규정.

섹션 구성:
A) 헤더 + "공항에서 압수당하지 마세요!" 경고 카드
B) 한국 출국 면세 한도 (중국 입국 기준):
   - 총액: 5,000위안 (약 100만원) 이하
   - 주류: 1병 (1.5L 이하, 알코올 12% 이상은 제한)
   - 담배: 400개비 (2보루)
   - 향수: 합리적 개인 사용량
C) 시각적 아이콘 카드 (한눈에 보기):
   - 술병 아이콘 + "1병"
   - 담배 아이콘 + "2보루"  
   - 향수 아이콘 + "적정량"
   - 돈 아이콘 + "5,000위안"
D) 액체류 기내 반입 규정:
   - 개별 용기 100ml 이하
   - 투명 지퍼백 1개 (20cm x 20cm)
   - 총 1L 이내
   - 면세점 구매 액체류: 밀봉된 STEB 백이면 OK
E) 자주 압수되는 것 TOP 5:
   1. 화장품 큰 통 (100ml 초과)
   2. 꿀/잼 (액체 분류)
   3. 음료수
   4. 치약 대용량
   5. 김치 국물
F) 출처: 인천국제공항공사, 중국 해관총서

주의: 중국 입국 기준은 중국 해관총서 규정 기준. "최신 규정은 해관총서 확인" 면책 추가.

### 5. 서비스탭/코스탭에서 진입점 연결
- 서비스탭의 travel 포켓(여행) 내부에 4개 가이드 링크 카드 추가
- 또는 TravelTab.jsx에 "출입국 가이드" 섹션 만들어서 4개 카드 배치
- 카드 터치 시 해당 가이드 컴포넌트를 바텀시트 or 전체화면으로 표시

### 테스트
- 4개 가이드 각각 렌더링 정상 확인
- 3개 국어 전환 작동
- 입국카드 번호 터치 -> 설명 표시
- 세금 환급 계산기 입력 -> 결과 출력
- 서비스탭 또는 TravelTab에서 4개 가이드 진입 가능
- 면책 문구 표시 확인
