# 위챗 미니프로그램 개발자 계정 등록 가이드

## 개요
위챗 미니프로그램을 개발하고 배포하기 위해서는 위챗 공식 개발자 계정이 필요합니다. 범범뻠 와이프 명의로 중국 신분증을 사용하여 등록하는 과정을 설명합니다.

## 필수 준비사항
- 중국 신분증 (범범뻠 와이프)
- 중국 휴대폰 번호
- 중국 은행 계좌 (미니프로그램 결제 기능 사용 시)
- 이메일 주소
- 사업자 등록증 (기업용 계정의 경우)

## 등록 단계

### 1단계: 위챗 공중호 플랫폼 접속
- URL: https://mp.weixin.qq.com/
- "立即注册" (즉시 등록) 클릭

### 2단계: 계정 유형 선택
- **개인 개발자**: 개인 신분증으로 등록 (추천)
- **기업 개발자**: 사업자 등록증 필요
- **정부/기관**: 정부 기관용

### 3단계: 기본 정보 입력
1. 이메일 주소 입력
2. 비밀번호 설정 (8자 이상, 영문+숫자+특수문자)
3. 이메일 인증 코드 확인

### 4단계: 실명 인증
1. **신분증 정보 입력**
   - 실명: 범범뻠 와이프 실명
   - 신분증 번호: 18자리 중국 신분증 번호
   - 신분증 사진: 앞면/뒷면 고화질 업로드
   
2. **휴대폰 인증**
   - 중국 휴대폰 번호 입력
   - SMS 인증 코드 확인
   
3. **얼굴 인증**
   - 위챗 앱을 통한 얼굴 스캔 인증
   - 신분증과 일치하는 본인 확인 필요

### 5단계: 계정 정보 완성
1. **기본 정보**
   - 개발자명/회사명
   - 연락처 정보
   - 주소
   
2. **개발 역량 확인**
   - 개발 경험
   - 기술 스택
   - 이전 프로젝트 (선택사항)

### 6단계: 계정 승인 대기
- 일반적으로 1-3일 소요
- 승인 완료 시 이메일 및 위챗 알림 수신

## 주의사항

### 보안 관련
- 계정 정보는 반드시 범범뻠 와이프 명의로 일관성 유지
- 2단계 인증 활성화 권장
- 비밀번호 정기 변경

### 법적 요구사항
- 중국 인터넷 규정 준수 필수
- 개인정보 보호법 준수
- 콘텐츠 검열 가이드라인 숙지

### 비용 정보
- 개인 개발자 계정: 무료
- 기업 계정 인증: 300위안 (약 50,000원)
- 결제 기능 활성화: 별도 수수료

## 등록 후 할 일

### 1. 개발 도구 설치
```bash
# 위챗 개발자 도구 다운로드
# https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
```

### 2. AppID 확인
- 등록 완료 후 고유한 AppID 발급
- 모든 개발/배포 과정에서 필요

### 3. 개발 환경 설정
- 개발자 도구 로그인
- 프로젝트 생성 및 설정

## 문제 해결

### 자주 발생하는 문제
1. **신분증 인증 실패**
   - 해상도: 300DPI 이상 권장
   - 파일 형식: JPG, PNG
   - 파일 크기: 2MB 이하

2. **휴대폰 인증 실패**
   - 중국 내 휴대폰 번호만 가능
   - VoIP 번호 사용 불가

3. **얼굴 인증 실패**
   - 조명이 충분한 환경에서 촬영
   - 신분증 사진과 일치하는 각도
